# 📈 자동 그리드 트레이딩 봇 (Auto Grid Trading Bot)

이 프로젝트는 그리드 트레이딩 전략을 기반으로 하는 고급 암호화폐 자동거래 봇입니다. 사용자 친화적인 GUI를 통해 실시간 모니터링 및 설정을 제공하며, 다양한 시장 상황에 대응할 수 있는 고급 기능을 포함하고 있습니다.

## ✨ 주요 기능

### 🎯 핵심 거래 전략
- **고급 그리드 트레이딩**: 상한선/하한선을 설정하여 가격 범위에 따른 그리드 개수를 자동으로 계산
- **추세 추종 매수 로직**: 그리드 도달 시 바로 매수하지 않고 하락 추세 확인 후 반등 시점에서 매수
- **추세 추종 매도 로직**: 그리드 도달 시 바로 매도하지 않고 상승 추세 확인 후 하락 시점에서 매도
- **0.1% 확인 버퍼**: 추세 변화를 확인하기 위한 정밀한 버퍼 시스템

### 🤖 지능형 자동 거래 시스템
- **고급 리스크 관리**: 
  - 🛡️ **보수적**: 15그리드, 30% 투자, -5% 손절
  - ⚖️ **안정적**: 20그리드, 50% 투자, -8% 손절
  - 🔥 **공격적**: 30그리드, 70% 투자, -12% 손절
  - ⚡ **극공격적**: 50그리드, 90% 투자, -15% 손절
- **스마트 투자금 분배**: 
  - 💡 **AI 기반 분배**: 코인별 성과, 변동성, 트렌드 분석으로 자동 분배
  - 📊 **실시간 재분배**: 매일 오전 9시 자동 재분배로 수익 최적화
  - 🎯 **코인별 한계**: BTC(20-50%), ETH(15-45%), XRP(10-40%) 비중 제한
  - 📈 **차트 기반**: 24시간 가격/거래량 데이터로 최적 분배 계산
- **다요소 성능 분석**: 시장 변동성, 트렌드 강도, 드로다운 비율 종합 분석
- **스마트 최적화**: 2단계 리스크 조정, 동적 파라미터 튜닝
- **포트폴리오 손절**: 전체 10% 이상 손실시 자동 정리
- **트레일링 스톱**: 최고점에서 설정% 하락시 수익 실현

### 📊 실시간 차트 및 UI
- **인터랙티브 차트**: 실시간 가격, 거래량, 기술지표 차트
- **📈 그리드 정보 실시간 표시**: 
  - 🕐 **(1시간/그리드10개)** 형식으로 현재 timeframe과 그리드 수 표시
  - 💰 **투자금 분배 상황**: 총투자금, 격당투자금, 분배비율 실시간 업데이트
  - 📊 **가격 범위 정보**: 상한선/하한선, 그리드 간격 명시
- **🎯 실시간 거래 마커**:
  - 📊 **매수 포인트**: 파란색 삼각형(↑) 마커, 크기 100, 흰색 테두리
  - 📊 **매도 포인트**: 빨간색 삼각형(↓) 마커, 크기 100, 흰색 테두리  
  - 🔵 **매수보류**: 연한 파란색 원형 마커, 크기 50, 투명도 0.8
  - 🔴 **매도보류**: 연한 빨간색 원형 마커, 크기 50, 투명도 0.8
- **차트 조작 기능**:
  - 🖱️ **마우스 드래그**: 좌클릭 + 드래그로 차트 이동
  - 🎯 **마우스 휠**: 포인터 중심 확대/축소
  - ⌨️ **키보드 단축키**:
    - `+`/`=`: 확대 | `-`: 축소 | `R`: 리셋
    - `←→↑↓`: 방향키로 차트 이동
- **네비게이션 툴바**: matplotlib 표준 도구 제공
- **실시간 호버 정보**: 차트 위 마우스 오버 시 거래 포인트 상세 정보 표시

### 📊 코인별 맞춤형 그리드 시스템
- **코인별 차별화**: 
  - 🥇 **BTC**: 20그리드, 7일 범위, 변동성x1.0 (안정적)

- **지능형 최적화**: 승률, 시장상황, 코인 특성 모두 고려
- **GUI 설정**: '코인별 설정' 탭에서 개별 맞춤 설정 가능

  - ⚡ **즉시 반영**: 거래 발생 시 로그와 차트에 동시 업데이트

  - 🔊 **5회 반복 TTS**: 극한 시장 상황 시 경고 메시지 5번 반복 알림

## 🎮 사용법

### 기본 거래 시작
1. **거래 모드 선택**: 
   - 🧪 **데모 모드**: 가상 거래로 안전하게 테스트
   - 💰 **실거래 모드**: 실제 자금으로 거래 (현재 보유 자산 자동 검색)
2. **코인 선택**: BTC, ETH, XRP 중 복수 선택 가능
3. **총 투자금 설정**: 
   - **데모 모드**: 가상 자금 입력
   - **실거래 모드**: 실제 보유 현금 기준으로 자동 설정
4. **자동모드 ON**: 리스크 모드 선택 (보수적/안정적/공격적/극공격적)
5. **투자금 분배**: AI가 각 코인별 성과 분석 후 자동 분배
6. **코인별 설정**: 설정 → 코인별 설정에서 개별 맞춤 설정 (선택사항)
7. **거래 시작 버튼** 클릭하면 선택된 모든 코인에서 동시 거래

### 차트 조작 방법

- **도구 사용**: 상단 네비게이션 툴바 활용

## 🧠 고급 트레이딩 로직

### 📊 그리드 트레이딩 전략
1. **동적 가격 범위 설정**: 
   - 📈 **timeframe 최적화**: 30분/1시간/2시간/4시간/12시간 중 최적 timeframe 자동 선택
   - 🎯 **코인별 맞춤 설정**: BTC(1시간), ETH(12시간), XRP(12시간) 등 각각 최적 분석 기간
   - 📊 **실시간 범위 갱신**: 시장 변동성에 따른 동적 가격 범위 조정
2. **지능형 그리드 개수 계산**: 
   - 💰 **투자금 기반**: 총 투자금과 수수료(0.05%)를 고려한 최적 그리드 개수 자동 계산
   - 🔄 **리스크별 조정**: 보수적(10-15개), 안정적(15-20개), 공격적(20-30개), 극공격적(30-50개)
   - 📈 **성과 기반 최적화**: 승률과 수익률 분석으로 1시간마다 자동 최적화

### 🎯 정밀 매수 로직 (Trend-Following Buy Strategy)
```
1단계: 그리드 가격 도달 감지
   ├─ 현재가 ≤ 매수 그리드 가격
   └─ "매수보류" 상태 진입 (연한 파란색 원형 마커 표시)

2단계: 추세 확인 및 대기 (0.1% 버퍼 시스템)
   ├─ 목표 하향 조정: 현재가 - (현재가 × 0.001)
   ├─ 추세 분석: 최근 5개 가격 데이터로 하락/상승/횡보 판단
   └─ 하락 추세 지속 시 계속 대기

3단계: 반전 확인 및 매수 실행
   ├─ 조건 충족: 현재가 > 확인가격(그리드가격 + 0.1%)
   ├─ 추세 전환: 하락 → 상승 or 횡보 감지
   ├─ 매수 실행: "데모 매수" or "실거래 매수"
   └─ 차트 표시: 파란색 삼각형(↑) 마커 + 실시간 업데이트
```

### 🎯 정밀 매도 로직 (Trend-Following Sell Strategy)  
```
1단계: 목표가 도달 감지
   ├─ 현재가 ≥ 매도 그리드 가격
   └─ "매도보류" 상태 진입 (연한 빨간색 원형 마커 표시)

2단계: 추세 확인 및 대기 (동적 목표 조정)
   ├─ 상승 지속 시: 목표가 상향 조정 (그리드 단위로 상승)
   ├─ 최고가 추적: highest_price 업데이트
   └─ 상승 추세 지속 시 계속 대기

3단계: 하락 확인 및 매도 실행
   ├─ 조건 충족: 현재가 < 확인가격(목표가 - 0.1%)
   ├─ 추세 전환: 상승 → 하락 or 횡보 감지  
   ├─ 매도 실행: "데모 매도" or "실거래 매도"
   └─ 차트 표시: 빨간색 삼각형(↓) 마커 + 실시간 업데이트
```

### 🔄 실시간 차트 마커 시스템
- **📊 매수 포인트**: 파란색 삼각형(↑) - 크기 100, 흰색 테두리
- **📊 매도 포인트**: 빨간색 삼각형(↓) - 크기 100, 흰색 테두리  
- **🔵 매수보류**: 연한 파란색 원형 - 크기 50, 투명도 0.8
- **🔴 매도보류**: 연한 빨간색 원형 - 크기 50, 투명도 0.8
- **⚡ 실시간 업데이트**: 거래 발생 시 즉시 차트에 마커 표시 및 캔버스 새로고침

### 다중 레벨 리스크 관리
- **보수적**: 15그리드, 30% 투자, -3% 급락감지, -5% 손절, 2% 트레일링
- **안정적**: 20그리드, 50% 투자, -5% 급락감지, -8% 손절, 3% 트레일링  
- **공격적**: 30그리드, 70% 투자, -7% 급락감지, -12% 손절, 4% 트레일링
- **극공격적**: 50그리드, 90% 투자, -10% 급락감지, -15% 손절, 5% 트레일링

### 코인별 맞춤 최적화
- **실시간 시장 분석**: 24시간 변동성, 12시간 트렌드, 6시간 거래량 분석
- **동적 그리드 조정**: 고변동성시 +30%, 강한트렌드시 +20%, 고거래량시 +15%
- **코인 특성 반영**: BTC(안정), ETH(대상), XRP(고변동) 별 다른 민감도

### 지능형 자동 최적화
- **다요소 성능 분석**: 승률, 평균수익, 변동성, 트렌드, 드로다운 종합 분석
- **2단계 리스크 조정**: 강한/온화한 상향/하향 조정 시스템
- **동적 파라미터 튜닝**: 시장상황에 따른 버퍼, 손절기준, 트레일링 자동 조정
- **코인별 최적화**: 각 코인의 리스크 프로필과 특성에 맞는 개별 최적화
